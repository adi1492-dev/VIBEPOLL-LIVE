# Speaker Presentation View Prompt

**User:**
```
The presenter dashboard is great for managing the poll, but it has too many small buttons to project on a big screen. Build a 'Speaker View' overlay that strips away the controls, makes the poll question massive, centers the charts, and puts a giant countdown timer and QR code on the right side. It should look like a professional slideshow slide.
```

**Context:**
This imaginary prompt demonstrates how the `isSpeakerView` state was conceived. The AI would design an alternative layout matrix in `PollPresenter.tsx` optimized for high-contrast, large-font viewing on projectors.
